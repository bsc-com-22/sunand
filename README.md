# Sun & Soil Foundation — Pure HTML/CSS/JS Version

This directory contains the entire website converted to pure HTML, CSS, and Javascript. It has no build dependencies and can be served from any web server or opened directly in a browser.

## Structure

- `index.html`: Home page
- `about.html`: About Us
- `programs.html`: Our Programs
- `projects.html`: Projects & Impact
- `team.html`: Our Team
- `partners.html`: Partners & Alignment
- `contact.html`: Contact Us
- `css/style.css`: Base styles, variables, and custom utilities.
- `js/main.js`: Interactivity (Header scroll, Mobile menu, Animations).
- `assets/`: Images and other static files.

## Technology Stack

- **HTML5**: Semantic structure.
- **Tailwind CSS**: Used via CDN for utility styling.
- **Vanilla Javascript**: For interactivity and animations.
- **Google Fonts**: Fraunces (Display) and Inter (Sans).
- **Lucide Icons**: SVG implementations.

## Features

- **Responsive Design**: Works on mobile, tablet, and desktop.
- **Scroll Animations**: Page elements reveal as you scroll.
- **Animated Counters**: Impact numbers count up when visible.
- **Sticky Header**: Background blurs and shadows on scroll.
- **Interactive Forms**: Contact form opens user's mail client with pre-filled content.
